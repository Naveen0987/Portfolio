## 🚀 Run Locally

Clone the project

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build production

```bash
npm run build
```
